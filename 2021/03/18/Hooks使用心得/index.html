Hooks使用心得

Posted by Lanyi Liu on 2021-03-18

Hook是React 16.8.0版本增加的新特性，主要目的是在函数组件中实现state以及其他特性。
三个常用Hook

  1. State Hook
    使得函数式组件也具有state状态，并且可以进行读写操作
    具体用法(ts版本)：
    const [xxx,setXxx] = useState(initValue)
    第一次声明用于初始化state变量以及赋予修改此变量的方法。
    即xxx为 state变量，initValue为xxx的初始值。 setXxx（）可以用于修改xxx。
    以下是几个示例（ts)

const [countDown, setCountDown] = useState(0)
const [timing, setTiming] = useState(false)
const [verificationCode, setVerificationCode] = useState<number | string>()

  1. Effect Hook
    目的是在函数式组件中也可以使用类式组件的生命周期特性（严谨说是模仿）

    useEffect(() => {

      //执行任何操作
  return （）=>{
  //执行一些收尾工作，比如清除定时器、取消订阅等
  }
 },[stateValue]）

    //注意[stateValue]位置所在数组，相当于监听某一个值的改变，当它改变时，执行函数体内的操作。如果不写数组，相当于componentDidupdate();如果写空数组，相当于componentDidMount();如果数组中写属性值，相当于componentwillUnmount()

  2. Ref Hook
    语法： const refContainer = useRef()
    相当于类式组件中的 React.createRef

心得： hook写法及其舒服便捷，个人认为是React发展的一个大趋势，函数式组件在书写方面比起类式组件简便很多，不需要在重复去写this，写构造器。setState（）更新数据的方法也十分好用。hook配合ts使用着实能将代码的质量拉高一大截。
